## üéØ Topics Covered

### CSS Fundamentals

#### Selectors
- **Universal Selector** (`*`) - Targets all elements
- **Type Selector** - Targets specific HTML elements
- **Class Selector** (`.class`) - Reusable styles
- **ID Selector** (`#id`) - Unique element styling
- **Grouping Selectors** - Apply same styles to multiple selectors
- **Chaining Selectors** - Combine selectors for specificity
- **Descendant Selectors** - Target nested elements

#### Box Model
- **Padding** - Space inside the element
- **Border** - Edge of the element
- **Margin** - Space outside the element
- Understanding how these affect element size

#### Layout Concepts
- **Block vs Inline** - Display property fundamentals
- **Flexbox** - Modern layout system
  - Growing & Shrinking (`flex-grow`, `flex-shrink`)
  - Axes (main axis & cross axis)
  - Alignment properties (`justify-content`, `align-items`)

### JavaScript Fundamentals

#### Basics
- **Variables** - `let`, `const`, `var`
- **Data Types** - strings, numbers, booleans, arrays, objects
- **Functions** - Declaration, expressions, arrow functions
- **Loops** - `for`, `while`, iteration

#### DOM Manipulation
- **DOM Basics** - Understanding the Document Object Model
- **Manipulation** - Selecting and modifying elements
- **Events** - Event listeners and handlers
- **Object Basics** - Properties and methods

## üí° Key Takeaways
- CSS selectors determine specificity and styling priority
- Box model is crucial for proper spacing and layout
- Flexbox simplifies complex layouts
- JavaScript brings interactivity to static HTML
- DOM manipulation connects JS to HTML elements
- Events make webpages interactive and responsive
